<interfaces>
Expected from 02-01:
```java
public interface RailPlusSdkAdapter {
    void initialize(Callback<Boolean> callback);
    void requestCharge(String cardId, int amount, Callback<ChargeResult> callback);
    void getBalance(String cardId, Callback<BalanceSnapshot> callback);
    SdkStatusSnapshot getStatus();
    void shutdown();

    interface Callback<T> {
        void onSuccess(T result);
        void onError(SdkErrorCode errorCode);
    }
}
```

`BridgeResponseFactory` from 02-01 is the only runtime payload builder. `NativeBridge` should call it for:
```java
String buildChargeSuccess(...);
String buildBalanceSuccess(...);
String buildStatusSuccess(...);
String buildError(...);
String buildReportAck(...);
```

The existing JS contract from app/src/main/assets/webview/index.html expects:
```javascript
RailBridge.requestCharge(JSON.stringify({ cardId, amount }));
RailBridge.getBalance(JSON.stringify({ cardId }));
RailBridge.getSdkStatus();
RailBridge.reportError(JSON.stringify({ code, message, context }));
window.onBridgeResult(resultJson);
```
